# Magick Tasks

## Server

### Setup
```
npm install
```

### Run backend server
```
npx ts-node src/server.ts
```
## Client

### Setup
```
npm install
```

### Compiles and hot-reloads for development
```
ng serve
```
Automatically open in browser
```
ng serve -o
```

### Compiles and minifies for production
```
ng build --prod
